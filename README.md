##Git使用方法：

* 先提交代码到本地仓库
* 再从本地仓库推送到远程仓库
* 提交此次的提交名称 git commit -m "提交说明"
* 提交代码到远程仓库 git push -u origin main
* 提交说明：s-年-月-日-时间，如：s-2024-01-01-10:00
